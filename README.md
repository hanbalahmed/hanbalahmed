# how a basic react app code looks like in the `App.js` file
## react 17 is current version (will update on later versions)

```js
 import './App.css';
 import logo from './logo.svg';
 
 function App() {
  return (
   <div className="App">
     <img src={logo} />
     
     <h1>React App</h1>
     
     <p>some text</p>
   </div>
  )
 }
 
 export default App;
```
